# 微信收藏文章 → Markdown 批量导出 · 完整操作指南

> 适用系统：**macOS** / **Windows** / Linux  
> 依赖：Python 3（系统自带即可），零额外安装

---

## 一、文件说明

| 文件 | 用途 |
|------|------|
| `clipboard_monitor.py` | 监听剪贴板，每复制一篇链接自动抓取标题 |
| `download_markdown.py` | 读取链接列表，批量下载为 Markdown |
| `export.sh` | Mac / Linux 一键脚本 |
| `step1_collect.bat` | Windows 收集链接脚本 |
| `step2_download.bat` | Windows 下载 Markdown 脚本 |

---

## 二、Mac 操作步骤

### 第 1 步：收集链接

打开终端（Terminal），进入本目录，运行：

```bash
python3 clipboard_monitor.py
```

终端显示「等待复制链接…」后，切换到微信：

1. 打开微信 → 左侧菜单点「收藏」
2. **右键**一篇收藏文章 → 点「复制链接」（菜单第一行）
3. 切换回终端，看是否显示 `✅ 标题`
4. 重复右键下一篇 → 复制链接
5. 全部完成后，按 **`Ctrl + C`**（Control 键 + C）

链接保存位置：**桌面 → `wechat_urls.txt`**

### 第 2 步：下载 Markdown

在同一个终端窗口，运行：

```bash
# 下载到桌面 wechat_md 文件夹
python3 download_markdown.py ~/Desktop/wechat_urls.txt ~/Desktop/wechat_md
```

### 第 3 步：打开文件夹

下载完成后，Finder 会自动弹出窗口。手动打开方式：

```bash
open ~/Desktop/wechat_md
```

或直接在桌面找到 `wechat_md` 文件夹，双击进入。

---

## 三、Windows 操作步骤

### 第 1 步：收集链接

双击运行 **`step1_collect.bat`**，或在 PowerShell / CMD 中：

```cmd
python clipboard_monitor.py
```

终端显示提示后，切换到微信：

1. 微信 → 收藏
2. 右键文章 → 复制链接
3. 切换回终端确认标题
4. 全部完成按 **`Ctrl + C`**

链接保存到：**桌面 → `wechat_urls.txt`**

### 第 2 步：下载 Markdown

双击运行 **`step2_download.bat`**，或在命令行中：

```cmd
python download_markdown.py %USERPROFILE%\Desktop\wechat_urls.txt %USERPROFILE%\Desktop\wechat_md
```

### 第 3 步：打开文件夹

```cmd
start "" "%USERPROFILE%\Desktop\wechat_md"
```

或直接桌面打开 `wechat_md` 文件夹。

---

## 四、后续使用（一条命令）

已经收集过链接后，下次只看新文章：

```bash
# 重新收集链接
python3 clipboard_monitor.py

# 下载 Markdown（覆盖旧文件）
python3 download_markdown.py ~/Desktop/wechat_urls.txt ~/Desktop/wechat_md

# 打开文件夹
open ~/Desktop/wechat_md          # Mac
start "" "%USERPROFILE%\Desktop\wechat_md"   # Windows
```

## 五、常见问题

| 问题 | 解决 |
|------|------|
| 标题显示「未能获取」 | 网络超时，Ctrl+C 后自动重试 |
| 链接显示「已跳过（重复）」 | 同一篇文章不会重复收集 |
| 所有标题都无法获取 | 检查网络是否正常，微信文章是否可公开访问 |
| 下载的 Markdown 没有图片 | 图片保留为远程链接，可后续加图片下载功能 |
| Windows 报 `python` 不是命令 | 安装 Python 3 并勾选「Add to PATH」 |
